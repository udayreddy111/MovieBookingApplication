# ğŸ”¹ 1. Git Configuration Commands (First Time Setup)

### Check Git version

```bash

```

### Set username

```bash

```


```bash

```

```bash

```

# ğŸ”¹ 13. Undo & Reset Commands (VERY IMPORTANT)

### Unstage file

```bash
git restore --staged file.txt
```

### Discard local changes

```bash
git restore file.txt
```

### Soft reset (keep changes)

```bash
git reset --soft HEAD~1
```

### Hard reset (delete changes)

```bash
git reset --hard HEAD~1
```
